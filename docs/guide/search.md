# 如何查询 SOP

## 方法一：全文搜索（推荐）

点击页面顶部的 **🔍 搜索 SOP** 输入框，或按下快捷键 <kbd>/</kbd>，然后输入关键词即可：

- 可以输入中文，例如「退款」「报销」「入职」
- 搜索会同时匹配标题和正文
- 回车直接跳转到第一个命中结果，方向键切换

::: tip 找不到想要的 SOP？
说明该流程还没有整理到知识库中。请联系对应分类的负责人补充，或参考 [新增或修改](./contributing.md) 自行提交。
:::

## 方法二：按分类浏览

从顶部导航栏选择你关心的业务分类，左侧会展开该分类下所有 SOP 列表：

1. 点击顶部 **SOP 分类**
2. 选择具体业务线（如「客户服务」）
3. 在左侧边栏选择具体流程

## 方法三：通过链接直达

每篇 SOP 页面的地址都是稳定的，可以在企业微信 / 钉钉 / 飞书 中直接贴链接给同事。

例如：`https://<你的域名>/customer-service/refund`

## 方法四：下载原始附件

部分 SOP 附带 Word / PDF / Excel 原件，可以在文末「附件」区域点击下载。
