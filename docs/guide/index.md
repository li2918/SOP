## 什么是 SOP？

SOP（Standard Operating Procedure，标准作业流程）是把一件事情「正确地做完」的分步说明。一份好的 SOP 应包含：

- **适用场景**：什么情况下该执行
- **角色分工**：谁做什么
- **操作步骤**：每一步的动作、输入、输出
- **异常处理**：出错时如何处理、找谁
- **关联资料**：参考表单、附件、系统链接

```
├── 人事行政      （入离职、考勤、采购）

```
